Tentang Website Ini

Latar belakang, tujuan, dan informasi tentang proyek website edukasi ini.

📖 Latar Belakang

Website ini dibuat sebagai proyek Ujian Akhir Semester (UAS) mata kuliah Pendidikan Agama Islam dengan tema "Kisah Para Nabi dan Rasul sebagai Teladan Hidup". Proyek ini bertujuan untuk menyajikan informasi tentang 15 Nabi dan Rasul secara interaktif, informatif, dan mudah dipahami oleh semua kalangan.

Kisah para Nabi dan Rasul bukan sekadar cerita sejarah, melainkan panduan nyata dalam menjalani kehidupan. Setiap kisah menyimpan hikmah mendalam yang relevan untuk menghadapi tantangan zaman modern.

🎯 Tujuan Website

📚

Edukasi Islam

Menyajikan kisah 15 Nabi & Rasul secara lengkap, akurat, dan mudah dipahami oleh semua kalangan.

💡

Inspirasi Hidup

Mengambil nilai-nilai keteladanan dari kisah para Nabi untuk diterapkan dalam kehidupan sehari-hari.

🎮

Pembelajaran Interaktif

Melalui fitur quiz dan timeline, pembelajaran menjadi lebih menarik dan tidak membosankan.

🛠️ Teknologi yang Digunakan

HTML5 CSS3 JavaScript (Vanilla) Google Fonts GitHub VS Code

Website ini dibangun dengan teknologi dasar web (HTML, CSS, JavaScript) tanpa framework, sehingga dapat dipahami dan dikembangkan oleh pemula sekalipun.
